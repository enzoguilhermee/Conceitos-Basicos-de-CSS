Pseudo-classes

a:link {...} -> Não visitei essa página anteriormente

a:visited {...} -> Visitei essa página anteriormente

a:hover {...} -> Estado do link ao passar o cursor sobre ele

a:active {...} -> Pressionar botão do mouse sobre determinado link

Listas

Listas normais (ul)

Em listas não ordenadas eu posso colocar:

list-style-type: square; (Quadrado em cada início de lista)

  • Item 1
  • Item 2
  • Item 3

Listas ordenadas (ol)

Em listas ordenadas eu posso colocar:

list-style-type: upper-alpha; (Lista com as letras alfabéticas maiúsculas A, B, C ...)

list-style-type: lower-alpha; (Lista com as letras alfabéticas minúsculas a, b, c ...)

list-style-type: upper-roman; (Lista com as números romanos maiúsculas I, II, III ...)

list-style-type: lower-roman; (Lista com as números romanos minúsculas i, ii, iii ...)

list-style-type: decimal; (Lista com os números padrão 1, 2, 3 ...)

  1. Item 1
  2. Item 2
  3. Item 3
